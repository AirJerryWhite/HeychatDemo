# 黑盒语音 - 机器人 DEMO
## 一、项目概述
本项目为黑盒语音机器人的演示示例，包含用 Python 和 Go 两种语言实现的 demo。其中，Python目录存放的是 Python 语言的 demo，Golang目录存放的是 Go 语言的 demo。
